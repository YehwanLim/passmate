# 취준OS - 디자인 아이디어

## 선택된 디자인 철학: **Modern Professional SaaS**

### 디자인 무브먼트
**Contemporary B2C SaaS** - 대기업 채용 담당자와 취준생 모두에게 신뢰감을 주는 현대적 전문성

### 핵심 원칙
1. **신뢰감 기반의 미니멀리즘**: 불필요한 요소를 제거하고 핵심 정보에 집중
2. **명확한 정보 계층**: 시각적 우선순위를 통해 사용자의 주의를 자연스럽게 유도
3. **프로페셔널한 우아함**: 차갑지 않으면서도 신뢰감 있는 톤 유지
4. **인간 중심의 공간**: 넉넉한 여백으로 답답함 없이 집중력 향상

### 색상 철학
- **Primary Blue** (`#1e40af` - Blue-700): 신뢰감, 전문성, 안정성의 상징. 대기업 이미지와 부합
- **Background**: 순백색(`#ffffff`) - 깔끔함, 신뢰감, 전문성
- **Accent**: 밝은 그레이(`#f3f4f6`) - 섹션 구분, 시각적 호흡감
- **Text**: 진한 슬레이트(`#1f2937`) - 높은 가독성, 전문적 톤

### 레이아웃 패러다임
- **비대칭 구조**: 왼쪽에 텍스트/입력, 오른쪽에 시각적 요소 (모바일에서는 스택)
- **섹션 간 명확한 구분**: 충분한 수직 여백(padding-y: 6rem~8rem)으로 각 섹션을 독립적으로 인식
- **카드 기반 결과 표시**: 결과 섹션은 깔끔한 카드 레이아웃으로 정보 조직화

### 시그니처 요소
1. **프로그레스 바**: 직무 적합도를 시각적으로 표현 (Blue 그래디언트)
2. **미니 배지**: "Coming Soon" 상태 표시 (회색 배경, 작은 텍스트)
3. **입력 포커스 효과**: 부드러운 파란색 테두리와 그림자로 상호작용성 강조

### 상호작용 철학
- **명확한 피드백**: 모든 버튼 클릭 시 로딩 상태 표시
- **스크롤 유도**: Hero 섹션의 CTA 클릭 시 부드러운 스크롤 애니메이션
- **폼 검증**: 실시간 피드백 없이 제출 시에만 검증 (간결함 유지)

### 애니메이션 가이드라인
- **입장 애니메이션**: fade-in + slight-up (duration: 600ms, easing: ease-out)
- **로딩 상태**: 부드러운 스피너 (Blue 색상)
- **호버 효과**: 버튼은 배경색 변화 + 그림자 강화 (transition: 200ms)
- **스크롤 애니메이션**: 부드러운 스크롤 (behavior: smooth)

### 타이포그래피 시스템
- **Display Font**: `Pretendard` 또는 `Inter` (700 weight) - 제목, 메인 카피
- **Body Font**: `Pretendard` 또는 `Inter` (400/500 weight) - 본문, 설명
- **계층 구조**:
  - H1: 48px, 700 weight (Hero 메인 카피)
  - H2: 32px, 600 weight (섹션 제목)
  - Body: 16px, 400 weight (본문)
  - Small: 14px, 400 weight (보조 텍스트)

### 브랜드 톤
- **신뢰감**: "대기업 현직 PM이 만든" - 전문성 강조
- **효율성**: "10초 만에" - 빠른 결과 제공
- **실용성**: "합격률을 높이세요" - 구체적 성과 약속
