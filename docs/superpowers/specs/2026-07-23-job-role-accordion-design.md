# 지원 직무 아코디언 설계

## 목표

지원 직무 목록의 세로 길이를 줄이면서도 사용자가 직무 카테고리와 세부 직무를 빠르게 찾을 수 있게 한다.

## 화면 동작

- 기획·PM, 마케팅·브랜딩 등 9개 카테고리를 접힌 행으로 기본 표시한다.
- 사용자가 카테고리 행을 누르면 해당 카테고리의 직무 버튼만 펼쳐진다.
- 여러 카테고리를 동시에 펼칠 수 있다.
- 처음 화면에서는 모든 카테고리가 접혀 있다.
- 직무를 선택하거나 해제해도 펼친 카테고리 상태는 유지한다.
- `+ 직접 입력`은 카테고리 목록 아래에 항상 한 번만 표시한다.

## 구현 경계

- 기존 `JOB_ROLE_CATEGORIES` 데이터와 단일 직무 선택 상태를 재사용한다.
- 프로젝트에 이미 있는 Radix 기반 아코디언 UI 컴포넌트를 사용한다.
- 직무 선택 값, 직접 입력, 이전 지원서 복원, 분석 API 요청은 변경하지 않는다.
- 검색·필터·모달 선택기는 추가하지 않는다.

## 검증

- 기본 상태가 모두 접힌 다중 선택 아코디언인지 테스트한다.
- 각 카테고리에 제목과 해당 직무 버튼이 렌더링되는지 테스트한다.
- 직무 선택 및 이전 지원서 복원 관련 기존 테스트를 유지한다.
- 타입 검사와 프로덕션 빌드로 호환성을 확인한다.
